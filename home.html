How It Works

SignMitra translates Gujarati into sign language in three simple steps

1

Input Text or Speech

Type Gujarati text using our keyboard or use the microphone for voice input

2

Word to SiGML Notation

Different words are converted to Hamnosys notation and then to SiGML code which is stored in Database.

3

Avatar animation using SiGML

Watch our 3D avatar perform the signs with natural, fluid movements
